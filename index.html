INTRODUÇÃO

O CSS é essencial para organizar a estrutura visual de um site, especialmente por meio das propriedades Display e Position. O Display define como os elementos se comportam e se relacionam no layout (incluindo Flexbox e Grid), enquanto o Position controla sua localização na página, permitindo ajustes mais precisos. O seminário aborda esses conceitos e suas aplicações na criação de layouts funcionais e responsivos.

CONCEITO DA PROPRIEDADE DISPLAY

A propriedade display é a responsável por determinar os tipos de exibição, tanto internas quanto externas, de um elemento.Ela é utilizada no CSS para especificar como um elemento HTML será apresentado na página, ou seja, como ele ocupaespaço e se organiza em relação aos outros elementos, seja como um bloco ou um elemento inline.sAlém disso, essa propriedade define o layout aplicado aos elementos filhos, como fluxo, grid ou flex.

Explicação detalhada dos valores

Block

O valor block faz com que o elemento seja exibido como um elemento de bloco, ocupando toda a largura disponível do container. Ele sempre começa em uma nova linha e cria uma quebra antes e depois dele. Além disso, é possível definir altura, largura, margem e padding normalmente.

Inline

O valor inline faz com que o elemento fique na mesma linha que outros elementos, sem quebrar a linha automaticamente. Ele ocupa apenas o espaço necessário para seu conteúdo. Elementos como e usam esse tipo de exibição.

Inline-block

O inline-block mistura características do inline e do block. O elemento fica na mesma linha que outros elementos, mas permite definir propriedades como largura, altura, margem e padding. Esse valor é muito usado quando se quer alinhar elementos lado a lado, mantendo controle de tamanho.

Flex

O valor flex transforma o elemento em um container flexível, utilizando o modelo de layout chamado Flexbox. Nesse modelo, os elementos filhos podem ser organizados em linha ou coluna e podem ser facilmente alinhados, distribuídos e redimensionados. É muito usado para criar layouts modernos e responsivos.

Inline-flex

Funciona como o flex, mas o container se comporta como um elemento inline, ele pode ficar na mesma linha que outros elementos. Internamente, os elementos filhos ainda são organizados usando o modelo Flexbox.

Grid

O valor grid ativa o CSS Grid Layout, que é um sistema de layout baseado em linhas e colunas. Ele permite criar estruturas mais complexas e bidimensionais para organizar elementos na página, sendo muito utilizado para layouts completos de sites.

Inline-grid

Funciona parecido com o grid, mas o elemento se comporta como inline, podendo ficar na mesma linha de outros elementos. Mesmo assim, seus filhos são organizados usando o sistema de grade do CSS Grid.

Flow-root

O valor flow-root cria um novo contexto de formatação de bloco, o elemento passa a controlar melhor o layout interno dos seus filhos. Ele é útil para evitar problemas com elementos flutuantes (floats) e manter a organização do layout dentro do container.

Table

O valor table faz com que o elemento se comporte como uma tabela HTML, semelhante ao elemento . Isso permite usar propriedades relacionadas a linhas, colunas e células para organizar o conteúdo.

Block flow

Esse valor usa a sintaxe moderna de dois valores do display. block define que o elemento se comporta como um bloco no layout externo, enquanto flow indica que os elementos internos seguem o fluxo normal do documento (layout padrão). Na prática, é equivalente ao comportamento padrão de muitos elementos de bloco.

Block flex

Nesse caso, o elemento é exibido como bloco externamente, mas seus elementos internos são organizados usando o Flexbox. Esse valor é equivalente ao uso tradicional de flex.

Block grid

Aqui o elemento se comporta como bloco no layout externo, enquanto os elementos internos são organizados usando CSS Grid. Na prática, tem o mesmo comportamento de grid.

Block flow-root

Esse valor cria um elemento que se comporta como bloco e também estabelece um novo contexto de formatação de bloco, semelhante ao flow-root. Ele é útil para controlar o layout interno e evitar interferência de elementos externos.

Inline flow

O valor inline flow define que o elemento se comporta como inline externamente, enquanto o conteúdo interno segue o fluxo normal do documento. Isso significa que o elemento fica na mesma linha que outros, mas organiza seus filhos no layout padrão.

Inline flex

Nesse caso o elemento é exibido como inline, porém seus filhos são organizados usando o Flexbox. Ele é equivalente ao valor tradicional inline-flex.

Inline grid

O elemento se comporta como inline, mas organiza seus elementos internos usando o sistema Grid. É equivalente ao valor inline-grid.

Inline flow-root

Esse valor faz com que o elemento seja inline externamente, mas crie um novo contexto de formatação interno, semelhante ao inline-block. Inclusive, inline-block é considerado equivalente a inline flow-root.

CONCEITO DA PROPRIEDADE POSITION

A propriedade position no CSS determina como um elemento HTML é posicionado no layout da página, estabelecendo sua relação com outros componentes. Ela permite controlar a localização do elemento, seja no fluxo normal do documento ou em locais específicos na tela, oferecendo maior controle sobre a organização e o layout do conteúdo.

Static

O elemento segue o fluxo normal do documento, e propriedades como top, left, e z-index não têm efeito.

Relative

O elemento é posicionado de acordo com o fluxo normal, mas pode ser deslocado em relação à sua posição original, mantendo o espaço que ocupava.

Absolute

O elemento é retirado do fluxo normal do documento e é posicionado em relação ao primeiro ancestral com um valor de `position` diferente de `static`.

Fixed

O elemento também é removido do fluxo normal, mas é posicionado em relação à janela do navegador, permanecendo fixo mesmo durante a rolagem da página.

Sticky

O elemento alterna entre `relative` e `fixed` dependendo da rolagem. Ele se comporta como `relative` até atingir uma posição definida, tornando-se fixo a partir desse ponto.
